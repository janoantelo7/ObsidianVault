# Conceptos
## Base de datos
Unha base de datos é un conxunto organizado de datos que se almacenan e se xestionan de maneira estructurada.
## Tabla
Unha tabla é unha estructura dentro de unha base de datos que organiza os datos en filas e columnas. Cada columna representa un capo de datos e cada fila representa unha tupla.
## Tupla 
Unha tupla é unha fila de datos nunha tabla de unha base datos relacional. Representa un só conxunto de valores que se corresponden coas columnas da tabla. Por exemplo, nunha tabla de "usuarios", unha tupla podría representar os datos de un único usuario (nombre, correo, edad, etc.).
## Columna (atributo)
Unha columna en unha tabla define o tipo de dato que contendrá, como enteiros, cadenas de texto, fechas, etc. Cada columna ten un nombre único dentro da tabla e representa un atributo dos datos almacenados.
## Consulta (query)
Unha consulta é unha instrucción SQL que se utiliza para recuperar, modificar ou manipular os datos dunha base de datos. As consultas poden ser do seguinte tipo:
- **`SELECT`**: seleccionar datos de unha tabla.
- **`INSERT`**: insertar datos dentro de unha tabla.
- **`UPDATE`**: actualizar os datos de un registro dunha tabla.
- **`DELETE`**: eliminar un registro de unha tabla.
