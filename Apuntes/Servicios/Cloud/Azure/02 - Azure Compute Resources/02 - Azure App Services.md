O servizo de Azure App Service é unha oferta de Plataforma como Servicio (PaaS) que nos permite crear, implementar e escalar de maneira sinxela aplicacións web, backends para apps móviles e API RESTful sen ter que gestionar a infraestructura subxacente.

Os App Services veñen definidos polos seguintes requisitos:
- **Tier de precios:** Free, shared, basic, standard, premium, premiumV2-3 e isolated.
- **Tamaño das VM:** Pequena, médiana ou grande.
- **Número das VM:** O número de VMs dentro do plan.
- **Región:** Geolocalización de Azure onde a nosa app é desplegada.

Os tipos de App Services que podemos contratar son os sequintes:
- **Shared:** Executa as apps na misma máquina virtual que outras aplicacións nun entorno multi-tenant.
- **Dedicated:** Executa as apps nun entorno aislado e dedicado para as nosas apps, pero segue compartindo rede coa Scale Unit.
- **Isolated:** Executa as nosas apps en un espacio de computo e rede dedicado exclusivamente a elas nun entorno de single-tenant.
